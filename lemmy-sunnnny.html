Lemmy သုံးနည်း

written June 02, 2024 in fediverse, social media, open source software

Lemmy သုံးနည်းမပြောခင် Lemmy ဆိုတာ ဘာလဲ သိအောင် အရင်ပြောချင်ပါတယ်။

Lemmy ဆိုတာက Activity Pud protocol လို့ခေါ်တဲ့ software တွေအချင်း စကားပြောလို့ရတဲ့ စကားကို သုံးထားတဲ့ ဗဟိုချုပ်ကိုင်မှုမရှိတဲ့ federated universe (Fediverse) ထဲက software တစ်ခုပါ။ Fediverse ထဲမှာ Twitter ပုံစံတူ Mastodon, Instagram ပုံစံတူ Pixelfed အစရှိသည့် software များစွာလည်းရှိပါတယ်။ Meta (Facebook) ရဲ့ နောက်ဆုံးထွက် Threads ဆိုတဲ့ app ကလည်း Fediverse ထဲက software တစ်ခုပါပဲ။

သူတို့တွေအကုန်လုံးက Activity Pud လို့ခေါ်တဲ့ စကားကို သုံးနေတဲ့အတွက် တစ်ခုနဲ့တစ်ခု ချိတ်ဆက်ပြောဆိုလို့ရပါတယ်။ ဆိုချင်တာက Mastodon မှာ အကောင့်ရှိရင် Lemmy ပေါ်က post ကို comment တွေဝင်ပေး၊ like တွေဝင်လုပ်လို့ရပါတယ်။

နောက်ပြီး ဗဟိုချုပ်ကိုင်မှုမရှိဆိုတာက volunteer အများစုက ကိုယ့်ပိုက်ဆံစိုက်ပြီး ကိုယ့်ဝါသနာပါရာ server (Fediverse ထဲမှာ instance လို့လဲသုံးနှုန်း) တွေ ထောင်ထားတာကိုဆိုလိုတာပါ။ ဥပမာ ani[dot]social ဆိုရင် anime အတွက်၊ programming[dot]dev ဆိုရင် programming သမားတွေအတွက်၊ literature[dot]cafe ဆိုရင် စာဖတ်ရတာကြိုက်သူတွေအတွက်ဆိုပြီး ထောင်ထားတာပါ။ Mastodon မှာဆိုရင်လည်း မြန်မာတွေအတွက်ဆိုပြီး burma[dot]social ဆိုပြီး ရှိပါတယ်။ အဲ့လို ကွဲထွက်နေပေမဲ့လည်း အပေါ်ကပြောခဲ့သလို Activity Pud ဆိုတဲ့ စကားတူ ပြောတဲ့အတွက် server တစ်ခုနဲ့တစ်ခုက ချိတ်ဆက်လို့ရနေပါသေးတယ်။ အကယ်၍ ani[dot]social မြန်မာမှာ ban ခံရရင်တောင် တစ်ခြား server တွေကနေနေပြီး ချိတ်ဆက်ပြောဆိုလို့ရပါသေးတယ်။

Lemmy က community driven software ပါ။ နောက်ကနေ ထောက်ပံပေးနေတဲ့ billion ချီတန်ဖိုးရှိတဲ့ company တွေလည်းမရှိပါဘူး။ ဝါသနာတူ နှစ်ယောက် အစကနေစုပြီး ပြုလုပ်ထားတဲ့ free and open source (FOSS) software ပါ။ FOSS software ဖြစ်တဲ့အတွက် ကိုယ့်က programming တတ်ရင် တတ်နိုင်တဲ့ဖက်က ဝင်ကူလို့ရပါတယ်။ Programming မတတ်ရင်တောင် interface တွေ၊ အသုံပြုနည်း စာရွက်စာတန်း တွေကို မြန်မာဘာသာ ပြန်ဆိုပေးလို့ရပါတယ်။ ငွေအားနဲ့လည်းကူညီလို့ရပါတယ်။ ဘယ်လိုကူညီရမလဲဆိုတာကတော့ join-lemmy.org မှာကြည့်ပါ။

Lemmy သုံးနည်း

Lemmy website ကို ဝင်ဝင်ချင်းတွေ့လိုက်ရတာကို front page လို့ခေါ်ပါတယ်။

ပုံပါ

A - ကတော့ Timeline လို့ခေါ်ပါတယ်။ Local ဆိုရင် ani[dot]social က post တွေ အကုန်ပဲပေါ်ပါမယ်။ Subscribe ဆိုရင် ကိုယ် follow လုပ်နေတဲ့ community တွေက post တွေပဲပေါ်ပါမယ်။ All ဆိုရင် ani[dot]social နဲ့ချိတ်ဆက်ထားတဲ့ server တိုင်းက post တွေပေါ်ပါမယ်။ ဆိုချင်တာက Technology server က post တွေကော၊ Programming server က post တွေကောပေါ်ပါမယ်။

B - ကတော့ sorting ပါ။ New ဆိုရင် တင်ထားတာ အသစ်ဆုံးဖြစ်တဲ့ post တွေအရင်ပေါ်ပါမယ်။ Active ဆိုရင် comment တွေပေး ဆွေးနွေးနေကျတဲ့ post တွေအရင်ပေါ်ပါမယ်။ Hot ဆိုရင် Upvote အများဆုံးရနေတာတွေအရင်ပေါ်ပါမယ်။

C - ကတော့ community name ပါ။ Lemmy မှာ community တွေရှာချင်ရင် !community@server[dot]name ဆိုပြီး အလုပ်လုပ်ပါတယ်။ ဥပမာ Anime Universe Myanmar ကိုရှာချင်ရင် !aum@ani.social ဆိုပြီး search bar မှာရှာရပါမယ်။ ! ပါတာ သတိပြုပါ။

D - ကတော့ Display name ပါ။ Username က အကောင့်ဖွင့်တဲ့အချိန်မှာထည့်ထားတာ ကိုပြောင်းလို့မရတော့ပါဘူး။ Display name ကိုတော့ ကိုယ့်ကြိုက်ရာ setting ထဲမှာ အလွယ်တကူပြောင်းလို့ရပါတယ်။

E - ကတော့ language ပါ။ Post တွေကို "ဗမာစာ" လို့ထားပြီးတင်ရင် setting ထဲမှာ "ဗမာစာ" ကို ရွေးမထားသူတွေ မမြင်ရတော့ပါဘူး။ ကိုယ့်ကြိုက်သလို language တွေကို setting မှာ filter လုပ်လို့ရပါတယ်။

F - ကတော့ thumbnail ပါ။

Post တွေရဲ့ ဘေးမှာ တွေ့ရတဲ့ ဒီဟာကိုတော့ sidebar လို့ခေါ်ပါတယ်။ အထိက အားဖြင့် community, instance နဲ့ ပတ်သက်တာတွေတင်ထားလေ့ရှိတယ်။ Rules တို့၊ guide၊ status တို့လိုပေါ့။

Sidebar ရဲ့ အောက်ဆုံမှာတွေ့ရတာကတော့ community, instance တွေရဲ့ info ပေါ့။ လူဘယ်နှစ်ယောက်သုံးလဲ၊ moderator တွေကဘယ်သူလဲ စတာတွေပါ။ Modlog မှာဆိုရင်တော့ moderator တွေရဲ့ action တွေကို တွေ့နိုင်ပါတယ်။ Mod တွေ ဘယ် post တွေကို ဖျက်ချလိုက်သလဲ စတာတွေပါ။

Post create လုပ်တာက လွယ်ပါတယ်။ Title, Body, Language, Community ကိုပဲ အထိကထားပါ။ Title ရှိရုံနဲ့တင် post တင်လို့ရပါပြီ။ Language က အပါ်မှာပြောခဲ့သလို ဗမာလိုရေးထားရင် "ဗမာစာ" လို့ထားပါ။ Community က ကိုယ်တင်ချင်တဲ့ community ကိုရွေးပါ။

Post Body က markdown support လုပ်ပါတယ်။ စာလုံး ကြီးချင်ရင်ကြီး၊ စောင်းချင်ရင်စောင်း လို့ရပါတယ်။ Spoiler တွေကို ဖုံးထားလို့ရတာတွေလည်းပါပါတယ်။ ကိုယ့်တိုင်သုံးကြည့်မှ ရင်နှီးသွားမှာပါ။ Comment တွေမှာလည်း markdown သုံးလို့ရပါတယ်။

NSFW ဆိုတာ က not safe for work လို့ဆိုလိုတာပါ။ သွေးထွက်သံယို၊ nudity post တွေတင်မယ်ဆိုရင် အမှန်ခြစ်ပေးပါ။ Spoiler post တွေတင်ရင်လည်း အမှန်ခြစ်ပေးပါ။

URL မှာကျ YouTube link တို့ဘာတို့ ထည့်တဲ့ဟာပါ။

Thumbnail မှာကျ frontpage မှာပြောခဲ့သလို သေးသေးလေးပါ်နေတဲ့ ပုံတွေပါ။ URL မှာ link ထဲခဲ့ရင်တော့ thumbnail က auto generate ဖြစ်ပါတယ်။

Profile setting ထဲမှာ အထိကကျတာက NSFW post တွေမြင်ချင်ရင် Show NSFW, Blur NSFW ကိုအမှန်ခြစ်ပါ။

Bot account တွေမြင်ချင်ရင်တော့ Show Bot account ကို အမှန်ခြစ်ပါ။

Language setting မှာကျ Ctrl ဖိပြီး language တစ်ခုထက်ပိုတာရွေးလို့ရပါတယ်။ Undetermined ကို ရွေးထားပါ။ Post creation မှာ ဗမာစာပါချင်ရင် ဒီမှာလာလုပ်ရပါမယ်။

Account လုံခြုံရေးအတွက် 2FA ကိုဖွင့်ပါ။

နောက်ပြီး Lemmy web version မှာ စာပို့လို့ရပါတယ်။ ဒါပေမယ့်သူက မလုံခြုံပါဘူး။ Server owner, @hitagi က အကုန်မြင်ရပါတယ်။ ဒါကြောင့် Matrix ဆိုတဲ့ chat app ကိုသုံးခိုင်းတာပါ။ သူလည်း ဗဟိုစနစ်မရှိတဲ့ Discord, Slack ပုံစံတူ chat app ပါ။ Matrix အကွက်မှာ ကိုယ့် username ဖြည့်လိုက်ရင် "Send secure message" ဆိုပြီးပေါ်နေပါမယ်။ ဒါကတော့ မလိုပါဘူးလို့မြင်ပါတယ်။

ဒီလောက်ပါပဲ။ နားမလည်တာတွေအောက်မှာမေးထားလို့ရပါတယ်။
